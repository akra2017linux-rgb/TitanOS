# Roadmap - TitanOS 2024-2025

## 🎯 Visão Geral

Este documento descreve o roadmap de desenvolvimento do TitanOS. As datas são aproximadas e podem mudar baseado em feedback da comunidade e recursos disponíveis.

## 📅 Fases

### 🚀 Fase 1: MVP (Q2 2024)
**Status**: Em Desenvolvimento

- [x] Base Arch Linux customizada
- [x] Kernel otimizado
- [x] KDE Plasma 6.0 com tema
- [x] Instalador Calamares
- [ ] Drivers NVIDIA/AMD/Intel básicos
- [ ] PipeWire e Wayland
- [ ] ISO Release v0.1.0

**Objetivos**:
- Distribuição estável e bootável
- Interface básica funcional
- Suporte a hardware comum
